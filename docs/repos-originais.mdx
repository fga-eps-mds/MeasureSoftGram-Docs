# Repositorios originais

Estes sao os repositorios base do MeasureSoftGram, sem indicacao de semestre no
nome. Eles representam as versoes historicas e canonicas das quais os
repositorios de cada ciclo (como os `2026.1-*`) derivam. Tambem pertencem a
organizacao [fga-eps-mds](https://github.com/fga-eps-mds).

- **[MeasureSoftGram-Core](https://github.com/fga-eps-mds/MeasureSoftGram-Core)**:
  versao base do nucleo de calculo do modelo de qualidade.
- **[MeasureSoftGram-Service](https://github.com/fga-eps-mds/MeasureSoftGram-Service)**:
  versao base do modulo Service (backend).
- **[MeasureSoftGram-CLI](https://github.com/fga-eps-mds/MeasureSoftGram-CLI)**:
  versao base da interface de linha de comando.
- **[MeasureSoftGram-Parser](https://github.com/fga-eps-mds/MeasureSoftGram-Parser)**:
  versao base do interpretador de dados de entrada.
- **[MeasureSoftGram-Front](https://github.com/fga-eps-mds/MeasureSoftGram-Front)**:
  versao base do frontend.
- **[MeasureSoftGram-Action](https://github.com/fga-eps-mds/MeasureSoftGram-Action)**:
  versao base da integracao com pipelines de CI.
- **[MeasureSoftGram-Helper-Action](https://github.com/fga-eps-mds/MeasureSoftGram-Helper-Action)**:
  acao auxiliar de apoio a integracao continua.
- **[MeasureSoftGram-Docs](https://github.com/fga-eps-mds/MeasureSoftGram-Docs)**:
  repositorio deste site de documentacao da comunidade.

:::note
Para o trabalho do ciclo atual, prefira os [repositorios ativos](./repos-ativos.mdx).
Os repositorios originais servem como referencia historica e ponto de partida.
:::
