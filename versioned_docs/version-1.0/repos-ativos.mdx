# Repositorios ativos

Os repositorios abaixo correspondem ao ciclo de desenvolvimento de 2026.1 e sao
os que estao em manutencao ativa. Todos pertencem a organizacao
[fga-eps-mds](https://github.com/fga-eps-mds) no GitHub.

- **[2026.1-MeasureSoftGram-Core](https://github.com/fga-eps-mds/2026.1-MeasureSoftGram-Core)**:
  nucleo do MeasureSoftGram, responsavel pelo calculo do modelo de qualidade.
- **[2026.1-MeasureSoftGram-Service](https://github.com/fga-eps-mds/2026.1-MeasureSoftGram-Service)**:
  modulo Service (backend), que armazena e manipula os dados da plataforma.
- **[2026.1-MeasureSoftGram-CLI](https://github.com/fga-eps-mds/2026.1-MeasureSoftGram-CLI)**:
  interface de linha de comando (`msgram`) para extrair e calcular metricas.
- **[2026.1-MeasureSoftGram-Parser](https://github.com/fga-eps-mds/2026.1-MeasureSoftGram-Parser)**:
  interpreta e normaliza os dados de entrada usados pelo Core.
- **[2026.1-MeasureSoftGram-Front](https://github.com/fga-eps-mds/2026.1-MeasureSoftGram-Front)**:
  frontend (interface web) para visualizacao dos resultados.
- **[2026.1-MeasureSoftGram-Action](https://github.com/fga-eps-mds/2026.1-MeasureSoftGram-Action)**:
  integracao com pipelines de CI para coletar e enviar metricas.
- **[2026.1-MeasureSoftGram-AI](https://github.com/fga-eps-mds/2026.1-MeasureSoftGram-AI)**:
  modulo de inteligencia artificial que estende as capacidades da plataforma.
- **[2026.1-MeasureSoftGram-Plugin](https://github.com/fga-eps-mds/2026.1-MeasureSoftGram-Plugin)**:
  plugin complementar do ecossistema MeasureSoftGram.
- **[2026.1-MeasureSoftGram-DOC](https://github.com/fga-eps-mds/2026.1-MeasureSoftGram-DOC)**:
  documentacao do projeto referente ao periodo de 2026.1 (mantida em MkDocs, nao
  confundir com este site de comunidade).
